## Features
- Four-legged robotic dog
- Front robotic arm
- Revolute joints
- Center of mass analysis
- Torque calculation
- Stable crawl gait

---

## Torque Calculation
- Volume: 60.48 cm³
- Estimated Mass: 74.99 g (0.075 kg)
- Force: 0.736 N
- Shoulder Link Length: 20 mm
- Required Torque: 0.01471 N·m
- Recommended Servo Motor: SG90 Micro Servo (1.8 kg·cm)
